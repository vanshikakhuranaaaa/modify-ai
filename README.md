🎧 MODIFY.AI
Intelligent Emotion-Based Music Companion
MODIFY.AI is an AI-inspired web application that detects user emotion via webcam interaction and recommends Spotify playlists aligned with the detected mood.

The project demonstrates adaptive music personalization using real-time input, modern animated UI design, and scalable architecture for future deep learning integration.

🚀 Features
📷 Webcam-based emotion simulation

🎵 Dynamic Spotify playlist recommendation

🌌 Futuristic animated UI

⚡ Real-time interaction

🔮 Extendable for ML-based emotion detection

🛠 Tech Stack
React.js | JavaScript | CSS Animations | Spotify Integration
